# Generate-Ts-Cli

使用：

```
git clone https://github.com/CroatiaParanoia/generate-ts-cli.git

cd generate-ts-cli

npm i

npm link

// 在当前目录初始化 generateTs.config.json
generateTs init

// 在当前目录下 ， 根据 generateTs.config.json 编译出ts
generateTs compile
同
generateTs compile -t ts

// 在当前目录下， 根据 generateTs.config.json 编译出js
generateTs compile -t js


```
